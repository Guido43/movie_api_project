This is some documentation concerning the API for the My Movie application

``
Request URL Response information
GET localhost:8080/movies A list of all movies is sent json list of movie names
GET localhost:8080/movies/film_info Data relavent to an individual movie description, genre, director and movie image
GET localhost:8080/movies/genres Data about movie genres returned to user Text descriptions of genres
GET localhost:8080/movies/directors Data about movie director returned to user Text file of director biography
POST localhost:8080/movies/users Message verification of register Allowing new users to register their details
PUT localhost:8080/movies/username Message verification of changes Allowing users to update their username information
POST localhost:8080/movies/add_movie Message verification of added movie Allowing users to add a movie to list of favorites
DELETE localhost:8080/movies/remove_movie Message verification of removed movieAllowing users to remove a movie from their list of favorites
DELETE localhost:8080/movies/remove_user Message verification of de-register Allowing users to de-register from the API
