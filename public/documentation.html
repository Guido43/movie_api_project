This is some documentation concerning the API for the My Movie application

``
Request URL Response information
GET localhost:8080/movies Enables client to obtain a list of all movies available Returns an Array of JSON objects including Title, Description, Image and information about the Movie, its Genre, Actors and the Directon in the following format.
[{ "Genre": {"Name": "",
"Description": "",
}
"Director" {"Name": "",
"Bio": "",
"Birth: "",
"Death: "",}
"_id": "",
"Title": "",
"Description": "",
"ImagePath": "",
"Actors": []
"Featured": Boolean }, ]
GET localhost:8080/users/:username Enables ability to search for the details of a user by their username JSON Object returned with the users' details in the following format:
{"_id": "",
"username": "",
"password": "",
"email", "",
"birthday": "",
"favoritemovies": ["",""], }
POST localhost:8080/users Enables client to create a new user with information to be stored JSON Object returned confirming the information entered in the following format:
{"username": "",
"password": "",
"email", "",
"birthday": "",
"favoritemovies": ["",""]
"_id": "", }
PUT localhost:8080/users/:username Enables client to update an element of the users current stored information; username, password, email or birthday. JSON Object returned confirming the updated information in the following format:
"{\n _id: new ObjectId(\"\"),
\n username: "",
\n password: "",
\n email: "",
\n birthday: 0000-00-00T00:00:00.000Z,
\n favoritemovies: [], } + Message TEXT This user's profile was updated"
DELETE localhost:8080/users/:username Enables client to delete a user currently stored. Message TEXT returned confirming the user was deleted in the following format:
'username' + 'user was deleted'
GET localhost:8080/movies/:Title Enables a client to search for a movie's details by its Title JSON Object returned with the Movie's full details in the following format:
{ "Genre": {"Name": "",
"Description": "",
}
"Director" {"Name": "",
"Bio": "",
"Birth: "",
"Death: "",}
"_id": "",
"Title": "",
"Description": "",
"ImagePath": "",
"Actors": []
"Featured": Boolean}
GET localhost:8080/movies/genres/:Genre Enables client to search for movies from a specific genre Returns an Array of JSON Objects of movies from a specific genre in the following format:
[{ "Genre": {"Name": "",
"Description": "",
}
"Director" {"Name": "",
"Bio": "",
"Birth: "",
"Death: "",}
"_id": "",
"Title": "",
"Description": "",
"ImagePath": "",
"Actors": []
"Featured": Boolean },]
GET localhost:8080/movies/directors/:Director Enables client to get the details of any movies from a specific directorReturns a JSON Object, or an Array of JSON Obects depending on whether the director has directed one or multiple movies in the database, returned in the following format:
[{ "Genre": {"Name": "",
"Description": "",
}
"Director" {"Name": "",
"Bio": "",
"Birth: "",
"Death: "",}
"_id": "",
"Title": "",
"Description": "",
"ImagePath": "",
"Actors": []
"Featured": Boolean },]
GET localhost:8080/movies/actors/:Actors Enables client to get the details of any movies starred in by specific actors Returns a JSON Array of Objects returning the details of the movies in which a certain actor has starred in the following format:
[{ "Genre": {"Name": "",
"Description": "",
}
"Director" {"Name": "",
"Bio": "",
"Birth: "",
"Death: "",}
"_id": "",
"Title": "",
"Description": "",
"ImagePath": "",
"Actors": []
"Featured": Boolean },]
GET localhost:8080/movies/director_name/:Director Enables client to recieve just the specific information about the named director, Name, Biography, Birth and Death dates. Returns a JSON Object of the relevant information in the following format:
{"Name": "",
"Bio": "",
"Birth: "",
"Death: "",}
GET localhost:8080/movies/genre_name/:Genre Enables client to recieve just the specific information about the named Genre, Name, Biography, Birth and Death dates. Returns a JSON Object of the relevant information in the following format:
{"Name": "",
"Description": "",}
POST localhost:8080/users/:username/movies/:movieID Enables a client to add a movie to a user's list of favorite movies Returns a JSON Obejct with the user's relevant information profile showing that the movie's id number was added to the Array of favorite movies in the following format:
{"_id": "",
"username": "",
"password": "",
"email", "",
"birthday": "",
"favoritemovies": ["",""]}
DELETE localhost:8080/users/:username/movies/:movieID Enables a client to remove a movie from a user's list of favorite movies Returns a JSON Obejct with the user's relevant information profile showing that the movie's id number was deleted from the Array of favorite movies in the following format:
{"_id": "",
"username": "",
"password": "",
"email", "",
"birthday": "",
"favoritemovies": ["",""]}
